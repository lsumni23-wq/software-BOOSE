Class RectCommand
Draws a rectangle in the BOOSE interpreter. Syntax: rect width,height
Syntax
C#
    public class RectCommand : IBOOSECommand
Remarks
The width and height can be numeric literals or expressions evaluated in the current BOOSEContext. The evaluated values are converted to integers before drawing the rectangle.
