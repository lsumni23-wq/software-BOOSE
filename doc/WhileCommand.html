Class WhileCommand
Represents a conditional loop that executes a block of commands repeatedly while the condition evaluates to true. Syntax: while condition ...wend
Syntax
C#
    public class WhileCommand : IBOOSECommand
Remarks
The conditionExpression is evaluated before each iteration. The loopBody contains the lines of code to execute in each iteration. Use RunProgram(string) to execute the loop body in the current context.
